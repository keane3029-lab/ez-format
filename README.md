# KEANE3029-LAB 💀
The official headquarters for the .EZ protocol. 
We don't do boring, we don't do bloat. We do AURA.

## Protocol Specs
- `.ez` files are key-value based (`KEY VALUE`).
- Delimiter: `>`.
- Tokens: `@anim` (visuals), `$exec` (system).
- Aura Audit: Enforced by GitHub Actions.
