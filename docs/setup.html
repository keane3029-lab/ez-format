SETUP

Double-click setup

One-time setup per machine so .ez files open like any normal file.

Windows

1
Open Command Prompt (not PowerShell) as Administrator.
2
cd into your ez-format folder.
3
Register the file type:
cmd
assoc .ez=EZFile
ftype EZFile=python "%CD%\ez_run.py" "%1"
Explorer may still show an "Open with" picker the first time — this is a Windows quirk, not a bug in the setup. If it happens, save a tiny open_ez.bat next to ez_run.py:
open_ez.bat
@echo off
python "%~dp0ez_run.py" %1
Then in the picker, choose "Look for another app on this PC", select open_ez.bat, and check "Always use this app".

macOS

1
Make the runner executable: chmod +x ez_run.py
2
macOS needs a small wrapper for Finder to open directly — create a .command file:
bash
echo 'python3 "'"$(pwd)"'/ez_run.py" "$1"' > run_ez.command
chmod +x run_ez.command
3
In Finder, right-click a .ez file → Get Info → Open with → choose run_ez.command.
